Курсовой проект: конструирование программы анализа функции

Выполнил: Солдатов В.И.

Руководитель: Минакова О. В.

Группа: бИЦ-251   

Основной модуль программы анализа функции содержит: функции для вычисления значения в точке, табулирования, поиска экстремумов, поиска аргумента по значению функции и вычисления производной, функцию журналирования и записи в файл рассчитанных значений
